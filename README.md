# 🎨 Uzfa_Creations - Art Portfolio Website

**Uzfa_Creations** is a creative platform designed to bridge the gap between art and its admirers. This website serves as a digital gallery to display various art forms, including sketches, paintings, and digital designs.

---

## ✨ Key Features
* **Art Gallery:** A beautiful layout to showcase high-quality images of artwork.
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views.
* **About the Artist:** A dedicated section for the artist's profile and vision.
* **Contact Integration:** A simple way for visitors to reach out for commissions or inquiries.

## 🛠️ Tech Stack Used
* **Platform:** WordPress (Localhost)
* **Database:** MySQL (phpMyAdmin)
* **Design:** Custom Theme & UI/UX focused layout.

---

## 📂 How to Run Locally
1. **Download** the repository as a ZIP or clone it.
2. **Database:** Create a new database in your localhost (phpMyAdmin) and import the `uzfa_creations.sql` file provided in this repo.
3. **Files:** Copy the `wp-content` folder into your fresh WordPress installation directory.
4. **Configuration:** Update your `wp-config.php` file with your local database name, username, and password.
5. **Login:** Access the dashboard via `/wp-admin`.
